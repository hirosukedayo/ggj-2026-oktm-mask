# 検証手順 (Walkthrough)

## 概要
スポット判定ロジックが正しく機能し、指定された条件（特定2箇所の撮影）でのみエンディングに到達できることを確認する。

## 手順

### 1. 準備
- `npm run dev` でサーバーを起動。
- ブラウザで `http://localhost:3000/game` にアクセス。

### 2. エンディングルートの確認
1. 画面上の以下の付近をクリック（タップ）して撮影を行う（順不同）。
    - **左上付近** (x:200, y:150 付近) -> Spot A
    - **右下付近** (x:760, y:400 付近) -> Spot B
2. 2枚撮影後、自動的にリザルト画面へ遷移することを確認。
3. リザルト画面に「エンディングへ進む」ボタン（またはそれに準ずる正解メッセージ）が表示されていることを確認。

### 3. ループ（失敗）ルートの確認
1. 「もう一度」ボタンでリセット。
2. 今度は**画面中央** (Spot C) と**全く関係ない場所**（右上端など）を撮影。
3. リザルト画面にエンディングボタンが表示**されない**ことを確認。
4. Spot C に対応するエピソードテキスト（例：「昔よく遊んだ空き地だ」）が表示されていることを確認。
5. 「もう一度思い出す」ボタンで撮影画面に戻れることを確認。

### 4. 何も映っていない場合
1. スポット座標から大きく外れた場所（四隅の端など）を2回撮影。
2. リザルト画面で「何も思い出せなかった」等のメッセージが表示されることを確認。
