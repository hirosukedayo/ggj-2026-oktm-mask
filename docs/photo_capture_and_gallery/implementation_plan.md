# 実装計画 - カメラ演出と撮影形状の変更

## 目標
- **レンズのブラー**: レンズ（ビューファインダー）内の映像にも「ぼかし」を入れる。ただし、背景よりは明るく視認性は保つ（「色は明るいがぼかし」）。
- **撮影形状の変更**: 円形から**正方形（スクエア）**に変更し、より「写真を撮っている」感覚を強める。
- **撮影演出**: クリック時に「カシャッ」となるようなフラッシュ/シャッター演出を追加する。

## 変更内容

### 1. `components/MaskCamera`
- **デザイン変更 (CSS/TSX)**:
    - `.lens` の `border-radius` を `50%` から変更し、角丸の四角形（カメラのファインダー風）にする。
    - **レンズ内ブラー**: `.lensInner img` (または親div) に `filter: blur(4px)` 程度を適用。
        - 背景は `blur(20px)` なので、それよりは弱くする。
    - **レティクル**: クリックを促すフォーカス枠のようなデザインに変更。
- **機能変更**:
    - `handleMouseMove` での追従は中心基準のまま。
    - 撮影 (`handleClick`) 時のキャプチャ範囲を、見た目の四角形に合わせて調整（既にロジックは矩形切り出しだが、見た目が円だったので整合性を取る）。
    - **Flash Effect**: 撮影した瞬間に画面（またはレンズ内）が一瞬白くなるアニメーションを追加。

### 2. `app/game/page.tsx`
- サイドバーのサムネイル表示も正方形に変更（`aspect-ratio: 1`, `border-radius: 0` または小）。

## 技術的詳細
- **フラッシュ演出**: React state `isFlashing` を使い、全画面またはレンズ上のオーバーレイを一瞬表示・フェードアウトさせる。
- **ブラー**: CSS `filter: blur()` を使用。
