# 実装計画 - 奥多摩2026ゲームジャム (Team B)

## 目標
マスクをテーマにしたゲームのコアメカニクスである「クリッピングマスク」機能を実装する。
Next.jsプロジェクトのセットアップを行い、基本的なカメラ/マスクコンポーネントを作成する。

## ユーザーレビューが必要な事項
特になし。

## 変更内容

### プロジェクト構成
- `app/`
  - `page.tsx`: タイトル画面（仮）
  - `game/`
    - `page.tsx`: ゲームメインコントローラー
    - `[sceneId]/page.tsx`: シーンごとの実装
- `components/`
  - `MaskCamera/`: マスク機能を持つカメラコンポーネント
    - `MaskCamera.tsx`: メインロジック
    - `MaskCamera.module.css`: スタイル

### コアメカニクス: クリッピングマスク
- CSS `clip-path` または SVGマスクを使用して、画像の大部分を隠弊する。
- ユーザー操作（ドラッグなど）により、マスク（または背景）を移動させ、隠された部分を探索できるようにする。

## 検証計画
### 手動検証
- `npm run dev` でサーバーを起動。
- ブラウザでアクセスし、マスクが正しく適用されているか確認。
- ドラッグ操作でビューが移動するか確認。
