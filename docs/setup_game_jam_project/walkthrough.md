# Walkthrough - カメラ演出と形状変更

## 変更内容
- **`components/MaskCamera`**:
    - **形状変更**: レンズ（ビューファインダー）と撮影範囲を**円形から正方形（角丸）**へ変更しました。
    - **ブラー調整**: レンズ内の画像にも軽度のブラー（`blur(3px)`）を適用し、レンズ外は `blur(40px)` で強くぼかしました。
    - **フラッシュ演出**: 撮影時に白いオーバーレイを一瞬表示するアニメーションを追加しました。
    - **効果音 (`New`)**: 撮影（クリック）時に `public/sounds/cam1.mp3` を再生するようにしました。
- **`app/game`**:
    - サイドバーの「Evidence」リストのサムネイルを正方形に変更し、スクエアフォーマットの写真を反映させました。
    - **リストアニメーション**: 新しい写真が追加される際、光りながらポップインする演出と、「NEW」バッジ表示を追加しました。
- **画像素材**:
    - `public/images/incident_02.png`: 「ウォーリーを探せ」風の街角画像を生成し適用しました。

## 検証結果 (想定)
- マウスカーソルに追従する四角い枠が表示されること。
- クリックすると「カシャッ」という音と共にフラッシュが光り、左側に写真がアニメーション付きで追加されること。
