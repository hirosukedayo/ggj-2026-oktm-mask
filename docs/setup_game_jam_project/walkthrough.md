# Walkthrough - レイアウトシフトの修正と演出調整

## 変更内容
- **`app/page.module.css`**:
    - `.buttonWrapper` を追加し、ボタン用の高さ（60px）を常に確保するようにしました。
    - `.hiddenState` を追加し、ボタン非表示時は `opacity: 0` で制御するようにしました。
    - フェードインアニメーションの時間を `2s` から `1.5s` に短縮し、より自然な「ふわっと感」に調整しました。
- **`app/page.tsx`**:
    - ボタンを条件付きレンダリングから常時レンダリングに変更。ラッパーで囲むことで、ボタン出現時にテキスト位置がズレる問題を解消しました。

## 検証結果 (想定)
- プロローグ画面が表示された瞬間からテキスト位置が固定されていること。
- 3秒後にボタンが定位置にふわっと現れ、その際にテキストが上下に動かないこと。
