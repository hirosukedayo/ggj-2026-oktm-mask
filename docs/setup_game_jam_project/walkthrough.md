# Walkthrough - 描画ズレの完全修正

## 変更内容
- **`components/MaskCamera`**:
    - コンテナの枠線 (`border: 4px`) を削除しました。
        - **原因**: `box-sizing: border-box` の環境下で枠線をつけると、内部の描画領域が狭まり（800px -> 792px）、指定サイズ(800px)で描画されるレンズ内の画像とズレが生じていました。
    - 前景画像（レンズ内・クリア領域）にも `object-fit: cover` を明示的に適用し、背景画像と描画ロジックを統一しました。

## 検証結果 (想定)
- 背景画像とレンズ内の画像が、ピクセル単位で完全に一致すること。
- 右端や下端に行ってもズレが発生しないこと。
